# 📈 한국투자증권 Open API를 활용한 자동매매 시스템

## 1. 프로젝트 개요
본 프로젝트는 한국투자증권(KIS) Open Trading API를 활용하여 파이썬(Python) 기반의 주식 자동매매 봇(Bot)을 개발한 결과물입니다. 
단순한 API 호출 테스트를 넘어, 실시간으로 주가를 감시하고 설정한 타겟 가격에 도달하면 즉시 모의투자 계좌로 시장가 매수 주문을 집행하는 무한 루프(Infinite Loop) 시스템을 구현했습니다.

## 2. 파일 구성 및 개발 단계
프로젝트는 기능별 모듈화 및 단계별 테스트를 거쳐 완성되었습니다.

* `test_current_price.py` : (1단계) OAuth 토큰 발급 및 삼성전자(005930) 현재가 조회 API(`GET`) 연동 테스트
* `test_buy_order.py` : (2단계) 모의투자 계좌 정보와 Body Payload를 활용한 현금 매수 주문 API(`POST`) 연동 테스트
* **`auto_trading_bot.py` : (최종) 실시간 시세 감시 및 조건부 자동 매수 집행 메인 로직**

## 3. 핵심 자동매매 로직 (`auto_trading_bot.py`)
최종 봇은 다음과 같은 알고리즘으로 작동합니다.

1. **인증 및 초기화**: 발급받은 App Key, App Secret을 통해 Access Token을 확보합니다.
2. **시세 감시 루프 (`while True`)**: 
   * `FHKST01010100` (주식 현재가 조회) API를 지속적으로 호출하여 타겟 종목의 가격을 모니터링합니다.
3. **방어 로직 (`time.sleep`)**:
   * 증권사 서버에 과도한 부하(디도스 공격 등)로 인식되어 IP가 차단되는 것을 방지하기 위해, 루프 내에 `time.sleep(1)`을 적용하여 안전한 API 호출 주기를 확보했습니다.
4. **조건 판단 및 주문 집행**:
   * 조회된 현재가가 사용자가 설정한 목표가(`TARGET_PRICE`) 이하로 떨어질 경우, 즉시 `VTTC0802U` (주식 현금 매수) API를 시장가로 호출하여 1주를 자동 매수합니다.
   * 중복 구매를 방지하기 위해 매수 성공 시 `break`를 통해 루프를 종료합니다.

## 4. 실행 및 테스트 결과
* **테스트 환경**: Python 3.x, `requests` 라이브러리 사용 (Google Colab 환경에서 구동 확인)
* **테스트 종목**: 삼성전자 (005930)
* **결과**: 봇 가동 즉시 현재가를 정상적으로 파싱해왔으며, 인위적으로 목표가를 상향 조정하여 테스트한 결과 모의투자 계좌로 정상적인 시장가 매수 주문(응답코드 0)이 들어가는 것을 확인했습니다.
